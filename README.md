# CrispyLauncher

This script is made to work on linux using the files from Doom + Doom 2 and Heretic + Hexen games on steam.

#Installation

To install this script, clone this repo using this command :

git clone https://github.com/tacgnol3/CrispyLauncher.git

chmod + x CrispyLauncher/crispy_launcher.sh



To run :

cd CrispyLauncher && ./crispy_launcher.sh
